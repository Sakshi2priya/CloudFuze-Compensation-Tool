HOW TO ADD THE CLOUDFUZE LOGO ON THE LOGIN PAGE
==============================================

1. Locate the CloudFuze logo image file you have (the one you shared in chat).

2. Copy or save that exact image into THIS folder:
   Compensation_Tool\assets\

3. Save or rename the file as one of these (the app looks for them in order):
   - cloudfuze_logo.png
   - cloudfuze_logo.jpg
   - CloudFuze_Logo.png

   Or keep any filename that contains "cloudfuze" and ends in .png or .jpg.

4. Restart or refresh the Streamlit app. The logo will appear at the top of the login page.

No changes are made to your image; the app simply displays the file you place here.
